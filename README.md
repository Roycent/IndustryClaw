# IndustryClaw

工业现场任务协同产品原型。

基于 React + Vite + TypeScript，界面定位为深色工业风的现场执行系统原型，围绕班次值守、工单执行、告警处置、交接台账、设备助手工作台展开。

## 页面结构

### 1. 生产总览

包含：

- 班次统计卡
- 当班待办
- 值守状态表
- 设备上下文面板
- 未结项与升级风险

### 2. 设备助手工作台

包含：

- 自然语言查询入口
- 任务链执行视图
- 数据源 / SOP / 工单 / 通知引用面板
- 当前任务项
- 已写入结果

### 3. 告警 / 工单中心

包含：

- 工单执行列表
- 告警列表
- 升级记录
- 筛选标签
- 批量派发 / 催办 / 升级 / 导出交接项按钮

### 4. 班组交接 / 任务历史

包含：

- 交接记录头信息
- 交接摘要
- 未结项表
- 接收确认表
- 异常时间线

## 文案与界面原则

当前版本已按工业软件界面方式整理：

- 减少说明性、讲解型文案
- 增加字段名、状态、表头、筛选、表格、记录列表、操作按钮
- 统一使用责任人、状态、截止时间、SLA、升级、确认、接收、回执、来源、更新时间等表达
- 页面标题尽量模块化，如“交接记录”“未结项”“升级记录”“设备上下文”“任务历史”

## 技术栈

- Vite
- React 18
- TypeScript
- CSS

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```bash
http://localhost:5173
```

## 构建

```bash
npm run build
```

构建产物输出到：

```bash
dist/
```

## 部署

推荐静态部署到 Vercel。

建议配置：

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

`vite.config.ts` 已配置 `base: './'`，可用于常规静态托管或子目录部署。
