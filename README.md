# IndustryClaw

班组长工作台。

保留 4 个视图：首页 / 当班总览、设备助手、工单中心、交接页。

当前交互：

- **催回执**：`待回执 → 已催办 → 催办升级`
- **分派**：补派责任人，工单转处理中
- **升级**：写入升级记录，状态转 `已升级`
- **加入交接**：未结项进入交接清单，待接班人签收
- **异常 / 工单 / 设备联动**：查看处置建议、责任链、接班人
- **操作留痕**：首页保留最近动作记录

## 技术栈

- Vite
- React 18
- TypeScript
- CSS

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```bash
http://localhost:5173
```

## 构建

```bash
npm run build
```

构建产物输出到：

```bash
dist/
```
