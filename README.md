# IndustryClaw

班组长值班工作台。

保留 4 个视图：首页、设备助手、工单中心、交接页。

当前界面重点：

- **首页**：用值班口径先看先处理、谁没回、哪个要升、哪条线被卡住
- **设备助手**：右侧详情改成决策字段，直接盯卡点、责任人、超时、下一步、影响范围
- **工单中心**：弱化流程展示，突出未接、未回、超时、要升级、要带班交
- **交接页**：去掉 OA / 文控感，改成白班先看、未稳设备、未闭环、接班责任人、几点前处理
- **操作**：继续保留分派、催回执、升级、带班交这些可点击动作

## 技术栈

- Vite
- React 18
- TypeScript
- CSS

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```bash
http://localhost:5173
```

## 构建
