# IndustryClaw

班组长工作台版本。

这是一个围绕**班组长单角色**的工业现场前端原型，保留 4 个视图：首页 / 当班总览、设备助手、工单中心、交接页。

当前版本新增了更接近真实操作的**可点击交互原型**：

- **催回执**：点击后工单回执状态会从 `待回执` 变为 `已催办`，再次触发可进入 `催办升级`
- **分派**：支持对重点工单执行补派，责任人与状态即时更新
- **升级**：点击后会把事项写入“升级记录”面板，工单状态同步改为 `已升级`
- **加入交接**：点击后事项会立即进入“交接页 / 交接列表”
- **点异常 / 点任务看右侧详情**：在设备助手中点击设备、工单、异常，右侧直接展示推荐动作和责任链
- **操作回放**：首页保留最近动作日志，方便演示原型状态变化

## 技术栈

- Vite
- React 18
- TypeScript
- CSS

```bash

```

```bash
http://localhost:5173
```
