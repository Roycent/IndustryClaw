# IndustryClaw

班组长工作台。

保留 4 个视图：首页、设备助手、工单中心、交接页。

当前界面重点：

- **首页**：只看本班最急事项、待分派、待回执、待升级、交接项
- **设备助手**：按卡点查看当前责任人、下一责任人、超时、当前动作
- **工单中心**：继续执行分派、催回执、升级、加入交接
- **交接页**：看必交项、待签收、已升级未闭环、接班确认
- **值班记录**：保留最近操作更新

## 技术栈

- Vite
- React 18
- TypeScript
- CSS

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

```bash
http://localhost:5173
```

## 构建

```bash
npm run build
```

产物输出：

```bash
dist/
```
